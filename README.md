# 🌿 NaturaSync - Gestão de Saúde Natural

O **NaturaSync** é um sistema de gestão clínica desenvolvido para integrar os conhecimentos de **Naturologia** e tecnologia. O projeto faz parte da minha formação em *Sistemas para Internet* e visto facilitador o acompanhamento de pacientes em terapias naturais e integrativas.

## 🚀 Tecnologias Utilizadas
- **Linguagem:** Python 3.12
- **Estrutura:** FastAPI / Uvicórnio
- **Banco de Dados:** SQLite (SQLAlchemy ORM)
- **Frontend:** HTML/CSS (modelos Jinja2)
- **Servidor:** Ubuntu Server 24.04 (Local em Imbé/RS)

## 🛠️ Como Executar
1. Clone o repositório.
2. Crie o ambiente virtual: `python3 -m venv venv`
3. Ative a venv e instale as dependências (requirements.txt).
4. Executar: `uvicorn main:app --recarregar`
